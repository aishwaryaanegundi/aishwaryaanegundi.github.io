About Me

Engineer. Researcher. Builder.

I'm a Senior Machine Learning Engineer at 7Learnings GmbH in Berlin, where I design and operate production ML systems — refactoring data pipelines to support multiple model families (XGBoost, TensorFlow DNNRegressors, Prophet), building fully automated weekly model update workflows using GitHub Actions and Google Cloud Run, and improving hyperparameter tuning efficiency through domain-knowledge-driven search space reduction.

Before that, I was a Research Assistant at DFKI Berlin, working on language model training, open-source LLM evaluation, and political bias detection in social media. My earlier research at the Max Planck Institute of Informatics focused on cross-platform analysis of state-sponsored trolls — published at KONVENS 2022.

My career started as an Application Developer at ThoughtWorks, where I built clean, tested, production software — a discipline that still shapes how I write ML code today.
